Detailed Description

Umbrella header for GoF++ patterns.

Author
Noah G. Wood (@NoahGWood)

Includes all 23 Gang of Four design patterns provided by GoF++, grouped into Creational, Structural, and Behavioral categories.

Example Usage

#include <gofpp.hpp>
// Now you can use any GoF++ pattern without extra includes
auto& s = gofpp::Singleton<MyClass>::Instance();
